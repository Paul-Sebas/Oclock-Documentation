# Vrai ou faux ?

Pour javascript, toutes les valeurs sont soit "truthy" (vraies) soit "falsy" (fausses).

Voici une liste des valeurs "falsy" en JavaScript :

- false
- 0
- "" (chaîne vide)
- null
- undefined
- NaN

Toutes les autres valeurs sont "truthy", y compris les objets vides, les tableaux vides et les chaînes non vides.
