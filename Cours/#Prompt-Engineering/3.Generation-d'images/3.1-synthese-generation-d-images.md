# Module 3 : Génération d’images -- Note de synthèse

## 1. Objectif du module

Comprendre comment “briefer” une IA d’images (DALL-E, Bing Image Creator, etc.) pour obtenir des visuels utiles, pas juste “jolis”.  
Tu apprends à décrire un sujet, un style, une ambiance et un usage final.

## 2. Idées clés à retenir

- Un bon prompt d’image décrit : le sujet, le style, la composition, la lumière, les couleurs, les détails.
- L’ordre et la clarté des éléments aident le modèle à prioriser ce qui est important.
- Il faut adapter le visuel à son support : réseaux sociaux, diapo, fiche pédagogique, etc.
- Modifier un seul paramètre à la fois t’aide à comprendre l’impact de chaque choix.

## 3. Bonnes pratiques

- Commencer par un prompt simple, puis ajouter progressivement du détail.
- Préciser le format ou l’usage (bannière LinkedIn, post Instagram, illustration pédagogique…).
- Garder une trace des prompts qui fonctionnent bien (ton “prompt book visuel”).
- Tester plusieurs styles pour voir ce qui sert le mieux ton message (réaliste, flat, cartoon…).

## 4. Prompts types à garder sous la main

```plain
[Paysage urbain]
Une illustration d’un paysage urbain moderne, vue en plan large, style aquarelle, lumière de fin de journée, couleurs chaudes, quelques passants et des cafés en terrasse, ambiance calme.

[Visuel réseaux sociaux]
Image pour un post Instagram au format carré sur le thème [SUJET], style flat design, couleurs vives, texte lisible “TEXTE COURT”, fond épuré, pensée pour être lisible sur mobile.

[Illustration pédagogique]
Schéma simple et clair expliquant [CONCEPT] pour des débutants, style illustration éducative, couleurs douces, icônes minimalistes, texte très lisible, fond blanc.
```

### Lien avec les exercices du module

Exos 1 à 3 te font pratiquer :

- jouer sur style / lumière / détails pour un paysage,
- adapter une idée à plusieurs formats de réseaux sociaux,
- transformer un concept (ex. cycle de l’eau) en image pédagogique.
