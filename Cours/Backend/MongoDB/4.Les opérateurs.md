# Principaux opérateurs

## Opérateurs de comparaison

```js
// $eq - ==
{ field: { $eq: value } }

// $ne - !=
{ field: { $ne: value } }

// $gt - >
{ field: { $gt: value } }

// $gte - >=
{ field: { $gte: value } }

// $lt - <
{ field: { $lt: value } }

// $lte - <=
{ field: { $lte: value } }

// $in - Vérifie si la valeur fait partie d'un tableau
{ field: { $in: [value1, value2, ...] } }

// $nin - Inverse de $in
{ field: { $nin: [value1, value2, ...] } }
```

## Opérateurs logiques

```js

// $and - Permet de combiner deux conditions pour récupérer des documents
{ $and: [ { expression1 }, { expression2 }, ... ] }

// $or - Pour donner une condition ou d'autres pour récupérer des documents
{ $or: [ { expression1 }, { expression2 }, ... ] }

// $nor - inverse de $or
{ $nor: [ { expression1 }, { expression2 }, ... ] }

// not - pour obtenir l'inverse de l'expression donnée
{ field: { $not: { $eq: value1 } } }
```

## Opérateurs de mise à jour

```js
// $set - met à jour les valeurs des propriétés ciblées
{ $set: { field1: value1, field2: value2, ... } }

// $unset - retire une propriété d'un document
{ $unset: { field: "" } }

// $inc - Incrémente une valeur d'un certain montant
{ $inc: { field: amount } }

// $mul - Multiplie la valeur d'un certain montant
{ $mul: { field: amount } }

// $rename - Renomme une propriété
{ $rename: { oldFieldName: newFieldName } }

// $push - Ajoute un élément à une liste (comparable au push en JS)
{ $push: { field: value } }

// $pull - retire un élément d'une liste selon une condition donnée
{ $pull: { field: { <condition> } } }
```
