# Principales fonctions

## Insertion

```sh
# Insérer un document
db.collection.insertOne({ field1: value1, field2: value2 })

# Insérer plusieurs documents
db.collection.insertMany([{ field1: value1 }, { field2: value2 }])
```

## Récupération

```sh
# Recherche plusieurs documents en fonction du paramètre passé
db.collection.find({ field: value })

# Recherche un seul document en fonction du paramètre passé
db.collection.findOne({ field: value })

# Recherche un document via son ObjectID
db.collection.findById(ObjectId("60c72b2f9af1b24d2c8b4567"))
```

## Mise à jour

```sh
# Met à jour un document. En premier paramètre, on passe le critère pour trouver le document souhaité, et en deuxième paramètre, on utilise l'opérateur `$set` pour indiquer la propriété et la valeur à mettre à jour.
db.collection.updateOne({ field: value }, { $set: { fieldToUpdate: newValue } })

# Met à jour plusieurs documents
db.collection.updateMany({ field: value }, { $set: { fieldToUpdate: newValue } })

# Remplace complètement un document
db.collection.replaceOne({ field: value }, { newField1: newValue1, newField2: newValue2 })
```

## Suppression

```sh
# Supprime un document selon un critère
db.collection.deleteOne({ field: value })

# Supprime plusieurs documents selon un critère
db.collection.deleteMany({ field: value })
```
